Welcome to my curated gallery of **Oh My Posh (OMP) themes**!  
This repository is a mix of:

- ✨ **Original creations** — designed from scratch with my own imagination.  
- 🌍 **Community-inspired themes** — borrowed and created by the community.  
- 🔀 **Hybrid blends** — mashups of multiple themes, combined into something uniquely mine.  

Whether you’re looking for a clean, minimal prompt or a vibrant, expressive one, this collection has a variety of styles to explore.

## ⚙️ Prerequisites

<details>
  <summary>Install Oh My Posh</summary>

  1. Follow the official installation guide: [Oh My Posh Docs](https://ohmyposh.dev/docs/installation)  
  2. Example for Windows (PowerShell):  
  
     ```bash
     winget install JanDeDobbeleer.OhMyPosh -s winget
     ```
     
  3. Example for Linux (bash/zsh):  
     ```bash
     curl -s https://ohmyposh.dev/install.sh | bash -s
     ```
  4. Example for MacOS
     ```bash
     brew install jandedobbeleer/oh-my-posh/oh-my-posh
     ```
     
</details>

<details>
  <summary>Install Nerd Fonts</summary>

  1. Download a Nerd Font from [Nerd Fonts](https://www.nerdfonts.com/font-downloads).  
  2. Install it on your system.  
     ```bash
     oh-my-posh font install meslo
     ```
  3. Set your terminal to use the Nerd Font.

</details>

> 💡 If you’re using a **minimal theme**, Nerd Fonts can be skipped.
